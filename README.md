# DH 2026 Pedagogy Poster Resources

Public-facing copies of selected materials for **How to Go Digital in American
Literary Studies** by Kohei Furuya, Aoyama Gakuin University.

This directory is the standalone GitHub Pages repository. Classroom source
files remain in their original locations and must not be edited as part of the
conference-publication workflow. Revise the copies in this repository instead.

## Included

- `poster/`: submitted A0 poster PDF, retained as a conference reference
- `course-materials/`: detailed English course syllabus, Weeks 1–12 English
  handouts, the final-paper assignment, and three English research-methods handouts
- `ai-generated-papers/`: English AI-generated papers produced with Codex and
  Claude Code, plus their creation or experiment conditions
- `assets/images/`: five screenshots of tools used in the course, including
  Historical BERT Explorer as an additional site resource

## Deliberately excluded

- raw or masked student-response data
- student-comment digests and annotation exports
- consent forms
- teacher-only operational notes
- model caches, corpora, application source trees, and other large files
- passwords, credentials, tokens, and MDX deployment configuration
- the Lexical Field Lab screenshot, because that tool was not used in the
  final course account

## Public reading copies

Links on `index.html` point to generated HTML reading copies so that headings,
tables, links, and code spans display correctly in a browser. The corresponding
Markdown files remain in the repository as editable source files.

Japanese teaching materials, local datasets, classroom-only application
source files, analysis scripts, and raw experiment outputs remain outside this
public repository.

## Suggested citation

Kohei Furuya, “How to Go Digital in American Literary Studies,” DH 2026
Pedagogy & Training SIG Poster Slam, Daejeon, 27 July 2026.

Local preparation does not publish these changes. Committing, pushing, and
changing GitHub Pages settings are separate steps.
